# Digital Tools

## Description  
This project is a modern and user-friendly web application designed to provide powerful tools and features for users. It helps improve productivity, simplify tasks, and deliver a smooth user experience.

## Technology  
- React.js  
- Tailwind CSS  
- DaisyUI  
- JavaScript  
- JSON  

## Features  
- Responsive user interface  
- Product add to cart  
- Dynamic data rendering from JSON
